【网盘智能识别助手】可以智能识别网页中选中文字（一般背景为蓝色）里的 网盘链接 和 提取码/密码，提示并自动填写提取码。

目前已支持识别：`✅百度网盘` `✅阿里云盘` `✅腾讯微云` `✅蓝奏云` `✅天翼云盘` `✅和彩云` `✅迅雷云盘` `✅123云盘` `✅360云盘` `✅115网盘` `✅奶牛快传` `✅城通网盘` `✅夸克网盘` `✅Chrome 扩展商店` `✅Edge 扩展商店` `✅Firefox 扩展商店`（安装成功后可以使用页尾的链接进行测试）**识别速度小于 1 毫秒**。
